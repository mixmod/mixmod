\section{{\sc mixmod} command line (expert using)}
%=================================================
%\input{Using_MixmodCommandLine.tex}
The procedure is the same that in quantitative data sets case. In
the input file, the number of modalities must be entered by using
{\em NbModality} keyword. Model types for qualitative data sets
are: Binary\_p\_E, Binary\_p\_Ek, Binary\_p\_Ej, Binary\_p\_Ekj,
Binary\_pk\_E, Binary\_pk\_Ek, Binary\_pk\_Ej, Binary\_pk\_Ekj.\\
Let consider the b\_toby.xem  input file in MIXMOD/EXAMPLES/
directory:

    Binary_p_Ej

    Binary_pk_Ej
    Binary_pk_Ek
    Binary_pk_Ekj
